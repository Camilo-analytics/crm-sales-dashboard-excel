# CRM Sales Performance Dashboard — Excel
![Dashboard](CRM%20Dashboard%20excel.png)

## Project Overview
Interactive Excel dashboard built from the CRM Sales Performance SQL analysis.
Transforms raw sales data (8,800 records) into executive-level visualizations
using Pivot Tables and Charts.

## Dataset
| File | Rows | Description |
|---|---|---|
| sales_pipeline_clean.csv | 8,800 | Sales opportunities and deal values |
| sales_teams.csv | 35 | Sales agents and regional assignments |

## Tools Used
- Microsoft Excel 2024
- Pivot Tables
- VLOOKUP (cross-table lookup)
- Bar Charts

## Key Findings
- **West region leads revenue** at $3.57M despite fewer deals than Central
- **GTX Pro is the top product** generating $3.51M in Won deals
- **Darcel Schlecht leads agents** by revenue at $1.15M
- **Win rate: 48.16%** — 4,238 Won out of 8,800 total opportunities
- **Pipeline health:** 1,589 Engaging + 500 Prospecting deals in progress

## Dashboard Structure
| Sheet | Content |
|---|---|
| dashboard | Executive overview — 4 charts |
| revenue by region | Pivot Table + Chart |
| revenue by product | Pivot Table + Chart |
| top agents | Pivot Table + Chart |
| pipeline_distribution | Pivot Table + Chart |

## Skills Demonstrated
- Pivot Tables for data aggregation
- VLOOKUP to join data across sheets
- Chart formatting and dashboard layout
- Number formatting ($M / $K scale)
